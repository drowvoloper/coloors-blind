This project is inspired by the Coloors project included in "The Creative Javascript Course" by @developedbyed

I wanted to add something different and show not just the original colors but also how they would be seen by color-blind people (specifically, protanopia, deuteranopia and tritanopia types). These colors are generated using the "color-blind" module by @skratchdot.

If you want to learn more about accessible charts, you can read "How to Use Color Blind Friendly Palettes to Make Your Charts Accessible".
